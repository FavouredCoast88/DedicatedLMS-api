# Dedicated News API
 
A REST API built with Node.js and Express.
 
## Features
 
- CRUD Operations
- JWT Authentication
- Validation Middleware
- Error Handling
- Axios External API Integration
- Morgan Logging
- Winston Logging
- Jest Testing
 
## Installation
 
npm install
npm start
 
## Endpoints
 
GET /articles
 
POST /articles
 
GET /articles/:id
 
PUT /articles/:id
 
PATCH /articles/:id
 
DELETE /articles/:id
 
GET /articles/search
 
POST /auth/login
 
GET /news
